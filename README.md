# Bucket

Bucket is an AI-powered adventure generator that helps users discover activities and places to explore in any location.

Users can create personalized bucket lists based on their:

* location
* budget
* group type
* adventure preferences

The application will later include photo challenges, maps, progress tracking, and mobile gameplay.

---

## 🗺️ Planned Features

* AI generated adventure lists
* User authentication
* Bucket list history
* Photo-based task completion
* Map visualization of activities
* Shareable public bucket lists
* Memory collage generator
* Flutter mobile application
